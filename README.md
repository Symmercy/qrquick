# qrquick
Are you tired of those qr code generators that have you watch more ads than an hour long youtube video? If so, then this is the site for you.

This is a qr code generater two-shotted by gemini, it has all the bells and whistles you would want, change bgcolor, sure, change foreground color, sure. Add your own logo to the middle, no problem.
It is completely web based, you can access it [here](https://symmercy.github.io/qrquick)
